# Best Properties Real Estate

## Overview

The **Best Properties Real Estate** website is an HTML5 and CSS3-based website that showcases a fictional real estate company and its featured residential properties. It provides visitors with company information, featured home designs, a bungalow floor plan, individual room pages, and a property application form while demonstrating the use of fundamental HTML5 elements, CSS styling, image maps, and web forms.

The website is designed with a clean layout, consistent navigation, and a simple user interface to provide an intuitive browsing experience while demonstrating the basic concepts of front-end web development.

---

## Features

- Multi-page website with consistent navigation
- Home page featuring company information and residential properties
- Bungalow floor plan using an HTML image map
- Individual pages for the dining room, living room, and master bedroom
- Property application form with various HTML form elements
- HTML5 semantic layout (header, navigation, section, and footer)
- Basic CSS styling for layout, typography, colors, and backgrounds
- Clean and organized user interface
